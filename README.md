This is an Admin Panel Template using Prisma and MySql configuration please check package.json .

## Getting Started

First, run the development server:

```bash
git clone...

npm install

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
# and
create your .env for database configuration
# and
create your .env.local for auth_URL and auth_secret configuration
```
